Smart Plant Irrigation System

This project is a Smart Plant Irrigation System developed using an ESP32 microcontroller along with sensors and IoT integration to automate plant watering and monitor soil conditions remotely.

Features

Monitors soil moisture level in real time

Automatically controls water pump based on soil condition

Prevents over-irrigation and water wastage

Remote monitoring through IoT platform

Reliable and low-maintenance system

Suitable for smart gardening and agricultural applications

Hardware Components

ESP32 Microcontroller

Soil Moisture Sensor – Monitors soil moisture level

Water Pump – Supplies water automatically when soil is dry

Relay Module – Controls pump operation

Power Supply – Provides power to the system

Connecting Wires & Tubes

Software Components

Arduino IDE – Used for programming ESP32

IoT Platform (Blynk / ThingSpeak) – For remote monitoring

Embedded C – Programming language
